## ğŸ› Troubleshooting

### Common Issues & Solutions

#### 1. MongoDB Connection Error
```
Error: connect ECONNREFUSED 127.0.0.1:27017
```
**Solutions:**
- Ensure MongoDB is running: `mongod`
- Check MongoDB service status
- Use MongoDB Atlas cloud URL
- Verify `MONGODB_URI` in `.env`

#### 2. Port Already in Use
```
Error: listen EADDRINUSE :::5000
```
**Solutions:**
```bash
# Windows
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# Linux/Mac
lsof -ti:5000 | xargs kill -9
```

#### 3. Module Not Found
```
Error: Cannot find module 'express'
```
**Solution:**
```bash
rm -rf node_modules package-lock.json
npm install
```

#### 4. CORS Errors
**Solution:**
- Verify backend CORS configuration allows `http://localhost:5173`
- Check frontend API URL is `http://localhost:5000`

#### 5. AI Features Not Working
**Solutions:**
- Verify `BIOMISTRAL_API_KEY` is valid
- Check API key quotas and limits
- Ensure internet connection
- Check API service status

#### 6. Email/SMS Not Sending
**Solutions:**
- Verify SMTP credentials (use Gmail App Password)
- Check Twilio credentials and phone number format
- Verify phone numbers in Twilio trial account
- Check API quota limits

#### 7. Image Upload Failing
**Solutions:**
- Ensure `uploads/` directory exists
- Check file size limits (Multer configuration)
- Verify file permissions

#### 8. JWT Authentication Errors
**Solutions:**
- Verify `JWT_SECRET` is set in `.env`
- Check token expiration
- Clear browser cookies/localStorage
